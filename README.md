本システムの動かし方
1. git clone https://github.com/riku310/new_face_emo.git
2. docker-compose build => 一度buildしたらもうしない（はず）
3. docker-compose up
4. http://localhost:8501/ にアクセスする

今後の展望
1. 起動するカメラを2台にする
2. 同時にふたりの表情を推定して，先に笑った方が負けになるゲーム性を追加する
3. ディレクトリの構成を改善する
